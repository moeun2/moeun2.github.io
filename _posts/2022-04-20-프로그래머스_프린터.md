## 나의 풀이

```python
def solution(priorities, location):
    answer = 0
    queue =  [(i,p) for i,p in enumerate(priorities)]
    while queue:
        cur = queue.pop(0)
        maxNum = max(queue, key = lambda x:x[1])
        
        if(cur[1] < maxNum[1]):
            queue.append(cur)
        else:
            answer += 1
            if(cur[0] == location):
                return answer
    return answer
```
