# 제로(Class 2+/Silver4 )

- [문제링크](https://www.acmicpc.net/problem/10773)


```python
k = int(input())
numbers = []
for i in range(k):
    number = int(input())
    if number != 0:
        numbers.append(number)
    else:
        numbers.pop(-1)
    print(numbers)
answer = sum(numbers)
print(answer)
```
